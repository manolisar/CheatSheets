Reading Files

Reading Options

Data Selection & Indexing
Column Selection
df['column'] Select single column
df[['col1', 'col2']] Select multiple columns
df.column Dot notation (if valid name)
df.loc[:, 'col1':'col3'] Column slice by name
Row Selection
df.iloc[0] First row by position
df.iloc[0:5] First 5 rows
df.loc['index_name'] Row by index label
df.iloc[-1] Last row
Conditional Selection
df[df['col'] > 100] Filter by condition
df[df['col'].isin(['A', 'B'])] Filter by list of values
df[(df['col1'] > 50) & (df['col2'] < 100)] Multiple conditions (AND)
df[(df['col1'] > 50) | (df['col2'] < 100)] Multiple conditions (OR)
Query Method
df.query('col > 100') Query with string expression
df.query('col1 > 50 and col2 < 100') Complex query conditions
df.query('col in @my_list') Use external variables
Advanced Filtering
# Sample data filtering sales_df = pd.read_csv('sales.csv') # High-value customers in specific regions high_value = sales_df.query( 'revenue > 10000 and region in ["North", "South"]' ) # Recent transactions recent = sales_df[ sales_df['date'] >= '2024-01-01' ] # Complex filtering filtered = sales_df[ (sales_df['age'] >= 25) & (sales_df['age'] <= 65) & (sales_df['income'] > sales_df['income'].median()) ]
Data Cleaning
Missing Data
df.isnull() Check for missing values
df.notnull() Check for non-missing values
df.isnull().sum() Count missing values per column
df.dropna() Remove rows with missing values
df.fillna(0) Fill missing values with 0
df.fillna(method='ffill') Forward fill missing values
Duplicates
df.duplicated() Check for duplicate rows
df.drop_duplicates() Remove duplicate rows
df.drop_duplicates(subset=['col1']) Remove duplicates based on specific columns
df.duplicated().sum() Count duplicate rows
Data Types
df.astype({'col': 'int64'}) Convert data types
pd.to_numeric(df['col']) Convert to numeric
pd.to_datetime(df['col']) Convert to datetime
df['col'].astype('category') Convert to category
String Cleaning
df['col'].str.lower() Convert to lowercase
df['col'].str.strip() Remove leading/trailing whitespace
df['col'].str.replace('old', 'new') Replace text
df['col'].str.contains('pattern') Check if contains pattern
