Excel Shortcuts Cheat Sheet

Essential keyboard shortcuts for Microsoft Excel productivity and efficiency

Basic Navigation

Cell Movement

Arrow Keys Move one cell
Ctrl + Arrow Jump to edge of data
Ctrl + Home Go to cell A1
Ctrl + End Go to last used cell
Page Up/Down Move one screen
Alt + Page Up/Down Move one screen left/right
Ctrl + G Go to specific cell

Worksheet Navigation

Ctrl + Page Up/Down Switch between worksheets
Alt + Tab Switch between workbooks
F5 Go to dialog

Selection

Basic Selection

Shift + Arrow Extend selection
Ctrl + Shift + Arrow Select to edge of data
Ctrl + A Select all
Ctrl + Shift + End Select to last cell
Shift + Space Select entire row
Ctrl + Space Select entire column
Ctrl + Shift + Space Select entire worksheet

Advanced Selection

F8 Extend mode on/off
Shift + F8 Add to selection
Ctrl + Click Select non-adjacent cells

Common Workflows

Create Pivot Table

  1. Select data range
  2. Alt + N + V (Insert Pivot Table)
  3. Choose location (OK)
  4. Drag fields to areas
  5. Customize as needed

Quick Data Analysis

  1. Select data range
  2. Ctrl + Q (Quick Analysis)
  3. Choose Charts, Formatting, or Totals
  4. Select desired option
  5. Apply to worksheet

Freeze Panes

  1. Select cell below/right of freeze point
  2. Alt + W + F + F (View → Freeze Panes)
  3. Or use Alt + W + F + R for rows only
  4. Alt + W + F + C for columns only

Pro Tips

Flash Fill: Ctrl + E to auto-detect patterns and fill data
Name Box: Click name box (left of formula bar) to quickly navigate to cell
Multiple Worksheets: Hold Ctrl while clicking sheet tabs to select multiple
Double-click Fill: Double-click fill handle to auto-fill down to last row of adjacent data
Ctrl + Click: Follow hyperlinks in cells without entering edit mode
Alt + Enter: Force line breaks within cells for multi-line text

Function Shortcuts

Common Functions:
=SUM(A1:A10) - Add range
=AVERAGE(A1:A10) - Average of range
=COUNT(A1:A10) - Count numbers
=COUNTA(A1:A10) - Count non-empty
=MAX(A1:A10) - Maximum value
=MIN(A1:A10) - Minimum value
=IF(A1>10,"Yes","No") - Conditional
=VLOOKUP(A1,B:D,2,FALSE) - Lookup
Date Functions:
=TODAY() - Current date
=NOW() - Current date and time
=YEAR(A1) - Extract year
=MONTH(A1) - Extract month
=DAY(A1) - Extract day
=WEEKDAY(A1) - Day of week number
Text Functions:
=LEN(A1) - Length of text
=LEFT(A1,3) - First 3 characters
=RIGHT(A1,3) - Last 3 characters
=MID(A1,2,3) - 3 chars starting at position 2
=UPPER(A1) - Convert to uppercase
=LOWER(A1) - Convert to lowercase
=TRIM(A1) - Remove extra spaces

Platform Differences

Mac Differences:
• Cmd instead of Ctrl for most shortcuts
• Fn + F4 for absolute references
• Cmd + Return for array formulas
• Option + Down Arrow for filter dropdown
