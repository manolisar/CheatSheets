Common Workflows

Basic Box Modeling

  1. Create → Box
  2. Right-click → Convert to Editable Poly
  3. Select polygon mode (4)
  4. Extrude faces (Ctrl+Shift+E)
  5. Inset for detail (Ctrl+Shift+I)
  6. Connect edges (Ctrl+Shift+C)

UV Unwrapping

  1. Select object → Modifier List → Unwrap UVW
  2. Select faces in polygon mode
  3. Edit UVs → Open UV Editor
  4. Flatten Mapping or Planar Map
  5. Relax by face angles
  6. Pack UVs for efficient layout

Quick Turbosmooth

  1. Select object
  2. Modifier List → Turbosmooth
  3. Set Iterations to 1-3
  4. Use Isoline Display for preview
  5. Collapse stack when final

Pro Tips

Isolation Mode: Alt+Q isolates selected objects for focused modeling
Quick Align: Alt+A for angle snap, great for architectural modeling
Modifier Stack: Always work non-destructively - keep base mesh editable
Reference Images: Use viewport backgrounds for modeling reference
Naming Convention: Use descriptive names - helps with complex scenes
Backup Often: Increment saves (Shift+Ctrl+S) prevent data loss
Customize UI: Save custom workspaces for different tasks
Xview: Use X-ray mode (Alt+X) for interior selections
