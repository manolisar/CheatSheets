Common Workflows

Basic Photo Retouching

  1. Open image (Ctrl + O)
  2. Duplicate background layer (Ctrl + J)
  3. Use Healing Brush (J) for blemishes
  4. Adjust levels (Ctrl + L)
  5. Apply Unsharp Mask filter
  6. Save for web (Ctrl + Alt + S)

Layer Mask Workflow

  1. Add layer mask (Layer menu)
  2. Select brush tool (B)
  3. Paint with black to hide
  4. Paint with white to reveal
  5. Use gray for partial transparency
  6. Alt + click mask to view

Selection to Path

  1. Make selection with any tool
  2. Go to Paths panel
  3. Click "Make work path from selection"
  4. Refine path with Pen Tool (P)
  5. Save path for future use

Pro Tips

Quick Mask: Press Q to enter Quick Mask mode for painting selections
Spacebar: Hold Space to temporarily switch to Hand tool
Caps Lock: Toggle precise cursors for painting tools
Alt + Click: Create clipping mask when between layers in panel
Shift + Tool: Cycle through tool variants in same group
Right-click: Access context menus for brush settings and options
Ctrl + Click: Load layer as selection from Layers panel
