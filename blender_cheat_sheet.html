üîÑ Quick Transform Workflow

  1. Select object (LMB)
  2. Press G/R/S for transform mode
  3. Press X/Y/Z to constrain axis
  4. Type number for precise value

üéØ Basic Modeling Workflow

  3. Select elements (1, 2, 3)
  4. Extrude faces (E)
  5. Add loop cuts (Ctrl+R)

üöÄ Quick Workflows

üè† Basic House Modeling

  1. Delete default cube (X)
  2. Add cube for base (Shift+A ‚Üí Mesh ‚Üí Cube)
  3. Scale up (S ‚Üí 2 ‚Üí Enter)
  4. Edit mode (Tab)
  5. Extrude top face (3 ‚Üí E ‚Üí Z ‚Üí 1)
  6. Inset roof (I ‚Üí 0.2)
  7. Extrude roof up (E ‚Üí Z ‚Üí 0.5)

üé≠ Character Animation Setup

  1. Import/create character mesh
  2. Add armature (Shift+A ‚Üí Armature)
  3. Position bones in Edit mode
  4. Parent mesh to armature (Ctrl+P)
  5. Pose mode for animation
  6. Set keyframes (I)

üé® Material & Texturing

  1. Select object
  2. Switch to Shading workspace
  3. Add new material
  4. Add Image Texture node (Shift+A)
  5. Connect to Base Color
  6. Load texture image
