Common Workflows

Basic Floor Plan

  1. Set units (UNITS command)
  2. Create walls with PLINE or LINE
  3. Use OFFSET for wall thickness
  4. Add doors/windows with BREAK
  5. Insert blocks for fixtures
  6. Add dimensions and text
  7. Create title block

Precise Drawing Setup

  1. Set drawing units (UNITS)
  2. Set drawing limits (LIMITS)
  3. Configure snap and grid (F7, F9)
  4. Set up layers (LA command)
  5. Create text styles (STYLE)
  6. Set dimension styles (DIMSTYLE)

Block Creation

  1. Draw the objects to include
  2. Type BLOCK command
  3. Specify insertion point
  4. Select objects
  5. Name the block
  6. Insert with INSERT command

Pro Tips

Command Aliases: Type just the first 1-2 letters of most commands (L for LINE, C for CIRCLE)
Right-Click Menus: Right-click for context-sensitive options during commands
Dynamic Input: F12 toggles dynamic input for coordinate entry near cursor
Selection Cycling: Ctrl+W cycles through overlapping objects
Quick Properties: Hover over objects to see quick properties panel
Transparent Commands: Prefix with apostrophe (') to use during other commands
Object Tracking: F11 enables object snap tracking for precise alignment
Command History: F2 opens text window to review command history
Quick Command Reference: Most commands have 1-2 letter aliases. Type the alias + Enter for faster workflow. Example: L = LINE, C = CIRCLE, M = MOVE, CP = COPY
