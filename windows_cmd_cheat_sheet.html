File & Directory Operations

Navigation

cd \ Go to root directory
dir List directory contents

Common Workflows

System Diagnostic

  1. systeminfo > system_info.txt
  2. dxdiag /t dxdiag_report.txt
  3. sfc /scannow
  4. chkdsk C: /f
  5. ipconfig /all > network_info.txt
  6. tasklist > running_processes.txt

Network Troubleshooting

  1. ipconfig /all (check IP configuration)
  2. ping 8.8.8.8 (test internet connectivity)
  3. ping gateway_ip (test local network)
  4. nslookup domain.com (test DNS resolution)
  5. ipconfig /flushdns (clear DNS cache)
  6. tracert destination (trace network path)
  7. netstat -an (check active connections)

File System Cleanup

  1. cd %temp% (navigate to temp folder)
  2. del /s /q *.* (delete temp files)
  3. sfc /scannow (scan system files)
  4. cleanmgr (run disk cleanup utility)
  5. defrag C: /o (optimize drive)
  6. chkdsk C: /f (check for disk errors)

Batch File Template

  1. @echo off
  2. title My Script
  3. echo Starting process...
  4. if not exist "folder" md folder
  5. copy source destination
  6. echo Process completed
  7. pause

Pro Tips

Tab Completion: Use Tab to auto-complete file and folder names
Wildcards: Use * for multiple characters, ? for single character
Command History: Press F7 to see command history in a window
Multiple Commands: Use && to run commands sequentially (cmd1 && cmd2)
Output Redirection: Use > to save output to file (dir > filelist.txt)
Error Handling: Use || for alternative commands (ping host || echo Failed)
Run as Admin: Right-click CMD and select "Run as Administrator"
Quick Edit: Enable QuickEdit in Properties for easy copy/paste
PowerShell Advantage: Use PowerShell for more advanced scripting and object manipulation
Batch Variables: Use %1, %2, etc. for command line parameters in batch files
