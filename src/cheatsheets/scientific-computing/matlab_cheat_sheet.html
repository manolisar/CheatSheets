Function Definition

Basic function:

    result = x^2 + y^2;
end
Multiple outputs:

    std_val = std(data);

Useful Tips

MATLAB Best Practices:
• Use semicolon (;) to suppress output
• Preallocate arrays for better performance
• Use vectorized operations instead of loops
• Comment your code with %
• Use meaningful variable names
Performance Tips:
• zeros(n,m) is faster than A = []
• A(end) accesses last element
• Use logical indexing: A(A>0)
• Profile code with tic/toc
