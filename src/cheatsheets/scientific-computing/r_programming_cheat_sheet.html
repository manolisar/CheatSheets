R Programming

Statistical computing, data manipulation with dplyr/tidyr, ggplot2 visualization, and R data analysis workflows

Basic Syntax

x <- value Assign value to variable
x = value Alternative assignment
print(x) Print variable value
ls() List objects in environment
rm(x) Remove object
rm(list=ls()) Clear all objects

Data Types

c(1,2,3) Create vector
list(a=1, b=2) Create list
data.frame(x=1:3, y=4:6) Create data frame
matrix(1:6, nrow=2) Create matrix
factor(c("a","b","c")) Create factor
array(1:12, dim=c(2,3,2)) Create array

Data Import/Export

read.csv("file.csv") Read CSV file
write.csv(df, "file.csv") Write CSV file
read.table("file.txt") Read table file
load("file.RData") Load R data file
save(df, file="data.RData") Save R data file

Data Exploration

head(df) First 6 rows
tail(df) Last 6 rows
str(df) Structure of data
summary(df) Summary statistics
names(df) Column names
dim(df) Dimensions
nrow(df) Number of rows
ncol(df) Number of columns

Data Indexing

df[1,] First row
df[,1] First column
df$column Select column by name
df[1:3, 2:4] Subset rows and columns
df[df$x > 5,] Filter rows by condition

dplyr Functions

library(dplyr) Load dplyr library
select(df, col1, col2) Select columns
filter(df, col1 > 5) Filter rows
arrange(df, col1) Sort by column
mutate(df, new_col = col1 * 2) Create new column
summarise(df, mean_x = mean(x)) Summarize data
group_by(df, category) Group data
df %>% filter(x > 5) %>% select(y) Pipe operations

tidyr Functions

library(tidyr) Load tidyr library
pivot_longer(df, cols) Reshape wide to long
pivot_wider(df, names_from, values_from) Reshape long to wide
separate(df, col, into = c("a", "b")) Split column
unite(df, new_col, col1, col2) Combine columns

ggplot2 Basics

library(ggplot2) Load ggplot2 library
ggplot(df, aes(x, y)) Initialize plot
+ geom_point() Add scatter plot
+ geom_line() Add line plot
+ geom_bar() Add bar plot
+ geom_histogram() Add histogram
+ geom_boxplot() Add box plot

ggplot2 Customization

+ labs(title = "Title", x = "X", y = "Y") Add labels
+ theme_minimal() Apply minimal theme
+ scale_x_continuous() Customize x-axis
+ facet_wrap(~group) Create subplots
+ aes(color = group) Color by group

Statistical Functions

mean(x) Calculate mean
median(x) Calculate median
sd(x) Standard deviation
var(x) Variance
cor(x, y) Correlation
lm(y ~ x, data = df) Linear regression
t.test(x, y) T-test
anova(model) ANOVA analysis

Control Structures

if (condition) { code } If statement
for (i in 1:10) { code } For loop
while (condition) { code } While loop
function(x) { return(x^2) } Define function
apply(df, 2, mean) Apply function to columns
lapply(list, function) Apply to list elements

String Operations

paste(a, b, sep = " ") Concatenate strings
nchar(string) String length
substr(string, start, stop) Extract substring
gsub("old", "new", string) Replace text
strsplit(string, "separator") Split string
