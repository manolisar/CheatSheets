Common Workflows

Basic Building Model:
1. Set up levels (LL)
2. Create grids (GR)
3. Draw walls (WA)
4. Add doors/windows (DR/WN)
5. Create floors (FL)
6. Add roof (RF)
7. Place components
View Setup:
1. Duplicate floor plan (VP)
2. Set view template (VT)
3. Adjust visibility (VG)
4. Add sections/elevations (S/EL)
5. Create 3D views (3D)
6. Set up sheets
Family Creation:
1. New family template
2. Create geometry
3. Add parameters
4. Test family types
5. Load into project
6. Place instances
Documentation:
1. Create schedules
2. Add dimensions (DI)
3. Place text/tags (TX/TG)
4. Set up sheets
5. Place views on sheets
6. Add title blocks

Professional Tips

BIM Best Practices:
• Use proper naming conventions
• Maintain model hierarchy
• Create project templates
• Use view templates consistently
• Regular model coordination checks
Performance Optimization:
• Close unnecessary views
• Use linked files for large projects
• Purge unused elements regularly
• Control detail levels appropriately
• Use worksets efficiently
Collaboration Tips:
• Establish model standards early
• Use central file workflows
• Coordinate regularly with team
• Document design decisions
• Maintain consistent schedules
