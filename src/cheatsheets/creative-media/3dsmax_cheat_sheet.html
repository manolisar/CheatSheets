Common Workflows

Basic Modeling:
1. Create primitive (Create panel)
2. Convert to Editable Poly
3. Enter sub-object mode (F1-F5)
4. Use modeling tools (Extrude, Bevel, etc.)
5. Add modifiers as needed
6. Apply materials and render
Animation Setup:
1. Set animation length
2. Enable Auto Key (N)
3. Move to start frame
4. Set initial keyframes
5. Move to end frame and transform
6. Preview animation (Spacebar)
Lighting Workflow:
1. Create lights (Ctrl + L)
2. Position and aim lights
3. Adjust intensity and color
4. Set shadow parameters
5. Test render (F9)
6. Refine lighting setup
Rendering Process:
1. Set up cameras (Ctrl + C)
2. Apply materials (M)
3. Configure render settings (F10)
4. Test render (F9)
5. Adjust and refine
6. Final render to file

Professional Tips

Modeling Efficiency:
• Use reference images for accuracy
• Work with proper topology in mind
• Use modifiers non-destructively
• Keep geometry organized with groups
• Regular saves and scene states
Performance Optimization:
• Use proxy objects for heavy scenes
• Freeze objects not being edited
• Hide unnecessary geometry
• Use appropriate viewport display modes
• Optimize polygon count for render needs
Animation Best Practices:
• Plan keyframes with timing sheets
• Use graph editor for smooth curves
• Apply secondary animation principles
• Use constraints for complex relationships
• Preview frequently during animation
