Common Workflows

Basic Building:
1. Draw floor plan with Rectangle (R)
2. Push/Pull walls to height (P)
3. Create door/window openings
4. Add roof structure
5. Apply materials (B)
6. Create scenes for different views
Component Creation:
1. Select geometry for component
2. Make component (Shift + G)
3. Name and set options
4. Edit component (Enter)
5. Exit component (Esc)
6. Use in Component browser
Precision Modeling:
1. Use Tape Measure (T) for guides
2. Type exact dimensions while drawing
3. Use arrow keys for axis locking
4. Utilize inference system
5. Check measurements with Entity Info
6. Use construction geometry
Site Modeling:
1. Import site plan/survey
2. Scale and position correctly
3. Create terrain with sandbox tools
4. Model existing conditions
5. Add proposed design
6. Create context and surroundings

3D Modeling Techniques

Complex Geometry:
• Use Follow Me for curved extrusions
• Combine multiple operations
• Work with intersecting geometry
• Use components for repeated elements
• Model at appropriate scale for detail
Organic Shapes:
• Start with basic form
• Add detail progressively
• Use Artisan plugin for subdivision
• Utilize sandbox tools for terrain
• Import/trace from photo references
Architectural Details:
• Model trim and moldings separately
• Use arrays for repetitive elements
• Create libraries of standard parts
• Use layers for different trades
• Maintain hierarchy with groups

Professional Tips

Modeling Best Practices:
• Start simple, add detail later
• Use groups and components extensively
• Keep raw geometry on Layer0
• Work at real-world scale
• Purge unused elements regularly
File Organization:
• Use consistent naming conventions
• Organize with layers and scenes
• Create templates for projects
• Use external references when possible
• Maintain component libraries
Performance Optimization:
• Hide unnecessary geometry
• Use simplified display for complex models
• Work with proxy geometry when needed
• Avoid excessive face counts
• Use components instead of groups for repeated geometry

Export & Presentation

2D Export:
• Set up scenes with proper views
• Adjust style settings
• Export to PDF or image format
• Use LayOut for documentation
• Include dimensions and annotations
3D Export:
• Clean geometry before export
• Check for reversed faces
• Organize with layers/groups
• Choose appropriate file format
• Test in target application
