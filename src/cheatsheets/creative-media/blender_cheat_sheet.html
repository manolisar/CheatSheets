Common Workflows

Basic Modeling:
1. Delete default cube (X)
2. Add primitive (Shift + A)
3. Enter Edit mode (Tab)
4. Select vertices/edges/faces (1/2/3)
5. Use modeling tools (E, I, Ctrl+R, K)
6. Return to Object mode (Tab)
UV Unwrapping:
1. Select object in Object mode
2. Enter Edit mode (Tab)
3. Select all faces (A)
4. Mark seams (Ctrl + E > Mark Seam)
5. Unwrap (U > Unwrap)
6. Switch to UV Editing workspace
Basic Animation:
1. Select object
2. Move to frame 1
3. Insert keyframe (I > Location)
4. Move to later frame
5. Transform object (G, R, S)
6. Insert keyframe (I)
7. Play animation (Spacebar)
Sculpting Setup:
1. Add mesh (Shift + A > Mesh > UV Sphere)
2. Enter Sculpt mode (Ctrl + Tab)
3. Enable Dyntopo (Ctrl + D)
4. Adjust brush size (F)
5. Sculpt with various brushes
6. Use Remesh (Ctrl + R) as needed

Professional Tips

Modeling Efficiency:
• Use modifiers for non-destructive editing
• Learn edge flow for good topology
• Use reference images for accuracy
• Keep geometry clean and manifold
• Use collections for scene organization
Performance Optimization:
• Use wireframe for heavy scenes (Z key)
• Hide objects not being worked on (H)
• Reduce subdivision levels for viewport
• Use simplified display for modifiers
• Enable backface culling in overlays
Animation Best Practices:
• Plan animations with timing sheets
• Use graph editor for smooth motion
• Apply principles of animation
• Use constraints for complex rigs
• Bake physics simulations when final
