Common Workflows

Basic Animation:
1. Create composition (Ctrl + Alt + N)
2. Import footage (Ctrl + I)
3. Add keyframes (Alt + P, R, S, T)
4. Apply easing (F9)
5. Preview (Spacebar or 0 on numpad)
6. Render (Ctrl + M)
Text Animation:
1. Create text layer (Ctrl + T)
2. Add text animators (Ctrl + Shift + T)
3. Set animation properties
4. Adjust timing and easing
5. Preview and refine
Masking Workflow:
1. Select layer to mask
2. Use pen tool (G) to create mask
3. Adjust mask properties (M, F)
4. Animate mask if needed
5. Use mask modes (Add, Subtract, etc.)
3D Animation:
1. Enable 3D for layers (Alt + Shift + C)
2. Create camera (Ctrl + Alt + Shift + C)
3. Set up lighting (Ctrl + Alt + Shift + L)
4. Animate in 3D space
5. Use 3D camera tools (C)

Professional Tips

Performance Optimization:
• Use proxies for heavy footage
• Pre-compose complex animations
• Cache work area when possible
• Disable effects during animation
• Use adjustment layers efficiently
Organization Best Practices:
• Name layers descriptively
• Use color coding for layer organization
• Pre-compose related elements
• Create project templates
• Use expressions for repeated animations
Motion Graphics Tips:
• Plan animations with timing sheets
• Use graph editor for smooth motion
• Apply secondary animation for realism
• Follow 12 principles of animation
• Create style frames before animating
