Network Troubleshooting Windows

Network diagnostic commands and tools for troubleshooting connectivity issues on Windows

Basic Network Information

ipconfig Display IP configuration
ipconfig /all Display detailed configuration
hostname Display computer name
systeminfo | findstr /i "network" Display network adapter info
getmac Display MAC addresses
wmic computersystem get name Get computer name via WMI

IP Configuration Management

ipconfig /release Release current IP address
ipconfig /renew Renew IP address from DHCP
ipconfig /flushdns Clear DNS cache
ipconfig /registerdns Refresh DNS registration
ipconfig /displaydns Display DNS cache contents
netsh int ip reset Reset TCP/IP stack

Network Interface Management

netsh interface show interface Display network interfaces
netsh interface ip show config Show IP configuration per interface
netsh interface set interface "Wi-Fi" enabled Enable network adapter
netsh interface set interface "Wi-Fi" disabled Disable network adapter
netsh wlan show profiles Show WiFi profiles
netsh wlan connect name="NetworkName" Connect to WiFi network

DNS Configuration

nslookup google.com DNS lookup
nslookup google.com 8.8.8.8 DNS lookup using specific server
netsh interface ip set dns "Local Area Connection" static 8.8.8.8 Set primary DNS server
netsh interface ip add dns "Local Area Connection" 8.8.4.4 index=2 Add secondary DNS server
netsh interface ip set dns "Local Area Connection" dhcp Set DNS to automatic (DHCP)

Connectivity Testing

ping google.com Test basic connectivity
ping -t google.com Continuous ping
ping -n 10 8.8.8.8 Ping with specific count
ping -6 ipv6.google.com IPv6 ping
tracert google.com Trace route to destination
pathping google.com Combination of ping and tracert

Port and Service Testing

telnet google.com 80 Test port connectivity
netstat -an Show all connections and ports
netstat -rn Display routing table
netstat -b Show executable associated with connections
netstat -o Show process ID for each connection
netstat -e Display Ethernet statistics

PowerShell Network Commands

Test-NetConnection google.com Test network connectivity (PowerShell)
Test-NetConnection google.com -Port 80 Test specific port
Get-NetAdapter Get network adapter information
Get-NetIPConfiguration Get IP configuration
Get-DnsClientCache Display DNS cache
Clear-DnsClientCache Clear DNS cache

WiFi Management

netsh wlan show profiles Show saved WiFi profiles
netsh wlan show profile name="ProfileName" key=clear Show WiFi profile details with password
netsh wlan delete profile name="ProfileName" Delete WiFi profile
netsh wlan show wlanreport Generate WiFi report
netsh wlan export profile key=clear Export all WiFi profiles
netsh wlan import profile filename="profile.xml" Import WiFi profile

ARP and Network Discovery

arp -a Display ARP table
arp -d Clear ARP table
arp -s 192.168.1.1 aa-bb-cc-dd-ee-ff Add static ARP entry
net view Display computers in workgroup
net use Display network connections
ping -a 192.168.1.1 Ping with hostname resolution

Windows Firewall

netsh advfirewall show allprofiles Show firewall status for all profiles
netsh advfirewall set allprofiles state on Enable firewall for all profiles
netsh advfirewall set allprofiles state off Disable firewall for all profiles
netsh advfirewall firewall show rule name=all Show all firewall rules
netsh advfirewall reset Reset firewall to default settings

Routing and Network Routes

route print Display routing table
route add 0.0.0.0 mask 0.0.0.0 192.168.1.1 Add default gateway
route delete 0.0.0.0 Delete default route
route add 10.0.0.0 mask 255.0.0.0 192.168.1.1 -p Add persistent route

Network Troubleshooting Tools

netsh int ip reset resetlog.txt Reset TCP/IP stack with log
netsh winsock reset Reset Winsock catalog
sfc /scannow System file checker
netsh int tcp set global autotuninglevel=normal Reset TCP autotuning
dism /online /cleanup-image /restorehealth Repair Windows image

Event Log and Monitoring

eventvwr.msc Open Event Viewer
perfmon.msc Open Performance Monitor
resmon Open Resource Monitor
tasklist /svc List processes and services
services.msc Open Services console

Network Shares and SMB

net share Display shared resources
net use \\server\share Connect to network share
net use * /delete Disconnect all network drives
net session Display active sessions
net files Display open files on server

Common Troubleshooting Workflows

Basic Network Issues:
1. ipconfig /all (check configuration)
2. ping 127.0.0.1 (test local stack)
3. ping gateway IP (test local network)
4. ping 8.8.8.8 (test internet)
5. nslookup google.com (test DNS)
6. tracert google.com (trace path)
DHCP Problems:
1. ipconfig /release
2. ipconfig /renew
3. Check DHCP service status
4. Test with static IP
5. Check network cable/WiFi
6. Restart network adapter
DNS Resolution Issues:
1. ipconfig /flushdns
2. nslookup google.com
3. Test different DNS servers
4. Check hosts file
5. netsh int ip reset
6. Restart DNS Client service
Network Performance Issues:
1. netstat -e (check error counters)
2. perfmon (monitor network utilization)
3. resmon (identify network usage)
4. Check for malware
5. Update network drivers
6. Test with different cable/port

Professional Tips

Network Troubleshooting Best Practices:
• Always start with ipconfig /all
• Test layer by layer (OSI model)
• Document all changes made
• Keep network drivers updated
• Use Event Viewer for detailed errors
Performance Optimization:
• Disable unused network adapters
• Configure proper DNS servers
• Monitor network utilization
• Update network adapter drivers
• Optimize TCP window scaling
Security Best Practices:
• Enable Windows Firewall
• Use strong WiFi passwords
• Disable unnecessary services
• Monitor network connections
• Keep Windows updated
