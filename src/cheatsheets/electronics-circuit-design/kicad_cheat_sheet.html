Design Rules & DRC

Design Rule Setup

Basic Rules
Setup ‚Üí Design Rules:

Track & Via:
- Minimum track width: 0.1mm (typical)
- Minimum via size: 0.2mm (typical)
- Via drill size: 0.1mm smaller than via size

Clearances:
- Track to track: 0.127mm (5 mil)
- Track to via: 0.127mm
- Track to pad: 0.127mm
- Via to via: 0.127mm

Solder mask:
- Mask clearance: 0.051mm (2 mil)
- Mask minimum width: 0.25mm
- Paste margin: -0.025mm (typical)

DRC (Design Rule Check)

Running DRC
Inspect ‚Üí Design Rules Checker

Common DRC errors:
- Track too close to track
- Via too close to via  
- Track too close to pad
- Drill too small
- Copper pour not connected
- Unconnected nets
- Duplicate footprints
- Missing courtyard

Error markers:
- White arrows point to violations
- Click marker to see error details
- Fix errors and re-run DRC

Net Classes

Class Management
Setup ‚Üí Design Rules ‚Üí Net Classes:

Default class:
- Applied to all nets unless specified
- Set basic track width and clearance

Power class:
- Wider tracks for power nets
- Larger vias for current handling
- Example: 0.5mm tracks, 0.6mm vias

Differential pairs:
- Controlled impedance routing
- Matched track width and spacing
- Example: 0.127mm tracks, 0.127mm gap

- Buried: Connect inner layers only

Common Workflows

New Project Workflow

Project Setup
1. File ‚Üí New ‚Üí Project
2. Choose template or blank project
3. Set project name and location
4. Open schematic editor (Eeschema)
5. Place components and create schematic
6. Run Electrical Rules Check (ERC)
7. Assign footprints to components
8. Generate netlist
9. Open PCB editor (Pcbnew)
10. Import netlist to PCB
11. Place components on PCB
12. Route tracks between components
13. Add copper fills/pours
14. Run Design Rules Check (DRC)
15. Generate output files (Gerbers, drill, etc.)

- Thickness: 1.6mm (standard)

Tips & Best Practices

Schematic Best Practices

Schematic Guidelines
  • Component placement: Left to right signal flow
  • Power rails: VCC at top, GND at bottom
  • Net labels: Use descriptive names (CLK, DATA, etc.)
  • Reference designators: U for ICs, R for resistors, etc.
  • Hierarchical sheets: Break complex designs into blocks
  • Documentation: Add notes and comments
  • ERC clean: Fix all electrical rule violations

PCB Layout Best Practices

Layout Guidelines
  • Component placement: Group related components
  • Ground plane: Use continuous ground plane
  • Power distribution: Wide tracks for power nets
  • High-speed signals: Keep traces short and straight
  • Differential pairs: Route parallel with equal length
  • Via stitching: Connect planes with vias
  • Thermal relief: Use for ground connections to large pours
  • Silkscreen: Clear component labels and polarity marks

Manufacturing Considerations

DFM Guidelines
Minimum dimensions (standard PCB):
- Track width: 0.1mm (4 mil)
- Via size: 0.2mm (8 mil) 
- Drill size: 0.1mm (4 mil)
- Spacing: 0.1mm (4 mil)

Solder mask:
- Minimum opening: 0.1mm
- Minimum web: 0.1mm between openings
- Expansion: 0.05mm around pads

Silkscreen:
- Minimum line width: 0.15mm (6 mil)
- Minimum text height: 1mm
- Keep away from pads and vias

Assembly considerations:
- Component orientation consistency
- Test points for debugging
- Reference designators visible
- Fiducial markers for pick-and-place
