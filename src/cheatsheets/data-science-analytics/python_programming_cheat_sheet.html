Functions

Basic function:
def greet(name):
    return f"Hello, {name}!"
Default parameters:
def add(a, b=0):
    return a + b
Variable arguments:
def sum_all(*args):
    return sum(args)
Keyword arguments:
def process(**kwargs):
    for key, value in kwargs.items():
        print(f"{key}: {value}")
lambda x: x**2 Lambda function
map(function, iterable) Apply function to items
filter(function, iterable) Filter items by function

Classes

Basic class:
class Person:
    def __init__(self, name):
        self.name = name
    def greet(self):
        return f"Hi, I'm {self.name}"
Inheritance:
class Employee(Person):
    def __init__(self, name, salary):
        super().__init__(name)
        self.salary = salary
@property Create property decorator
@staticmethod Static method decorator
@classmethod Class method decorator

Exception Handling

Try-except block:

    risky_operation()
except ValueError as e:
    handle_error(e)
except Exception as e:
    handle_unexpected(e)
finally:
    cleanup()
raise ValueError("message") Raise custom exception
assert condition, "message" Assert statement
