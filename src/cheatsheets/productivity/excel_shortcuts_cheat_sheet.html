Excel Shortcuts

Essential keyboard shortcuts, formulas, and functions for Excel productivity

Navigation Shortcuts

Ctrl + Home Go to cell A1
Ctrl + End Go to last used cell
Ctrl + G Go to specific cell
Ctrl + Arrow Jump to edge of data
Ctrl + Page Up/Down Switch between sheets
F5 Go To dialog

Selection Shortcuts

Ctrl + A Select all / current region
Ctrl + Space Select entire column
Shift + Space Select entire row
Ctrl + Shift + Arrow Select to edge of data
Ctrl + Shift + End Select to last used cell
Shift + Click Extend selection

Editing Shortcuts

F2 Edit active cell
Ctrl + Z Undo
Ctrl + Y Redo
Ctrl + C Copy
Ctrl + V Paste
Ctrl + X Cut
Delete Clear cell contents
Ctrl + Enter Fill selected cells

Data Entry Shortcuts

Ctrl + D Fill down
Ctrl + R Fill right
Ctrl + ; Insert current date
Ctrl + Shift + ; Insert current time
Alt + Enter Line break within cell
F4 Repeat last action

Formatting Shortcuts

Ctrl + B Bold
Ctrl + I Italic
Ctrl + U Underline
Ctrl + 1 Format Cells dialog
Ctrl + Shift + $ Currency format
Ctrl + Shift + % Percentage format
Ctrl + Shift + # Date format
Ctrl + Shift + & Add border

File Operations

Ctrl + N New workbook
Ctrl + O Open file
Ctrl + S Save
F12 Save As
Ctrl + P Print
Ctrl + F2 Print Preview
Ctrl + W Close workbook

Formula Shortcuts

= Start formula
F4 Toggle absolute/relative ($)
Ctrl + Shift + Enter Array formula
Ctrl + ` Show formulas
F9 Calculate now
Shift + F3 Insert function dialog
Ctrl + Shift + A Insert function arguments

Data Operations

Ctrl + T Create table
Ctrl + Shift + L AutoFilter toggle
Alt + ↓ Open filter dropdown
F11 Create chart (new sheet)
Alt + F1 Create embedded chart
Alt + D + S Sort dialog

Basic Math Functions

SUM(A1:A10) Sum of range
AVERAGE(A1:A10) Average of range
COUNT(A1:A10) Count numeric values
COUNTA(A1:A10) Count non-empty cells
MAX(A1:A10) Maximum value
MIN(A1:A10) Minimum value
ABS(-5) Absolute value → 5
ROUND(3.14159, 2) Round to 2 decimals → 3.14

Text Functions

LEN(A1) Length of text
UPPER(A1) Convert to uppercase
LOWER(A1) Convert to lowercase
PROPER(A1) Proper case
TRIM(A1) Remove extra spaces
LEFT(A1, 3) First 3 characters
RIGHT(A1, 3) Last 3 characters
MID(A1, 2, 3) 3 chars starting at position 2

Date & Time Functions

TODAY() Current date
NOW() Current date and time
YEAR(A1) Extract year from date
MONTH(A1) Extract month from date
DAY(A1) Extract day from date
WEEKDAY(A1) Day of week (1-7)
DATEDIF(A1, B1, "D") Days between dates

Logical Functions

IF(A1>10, "High", "Low") Simple IF condition
AND(A1>0, B1>0) Both conditions true
OR(A1>10, B1>10) Either condition true
NOT(A1>10) Reverse logical value
ISNUMBER(A1) Check if cell is number
ISTEXT(A1) Check if cell is text
ISBLANK(A1) Check if cell is empty

Lookup Functions

VLOOKUP(A1, B:E, 3, FALSE) Exact match lookup
HLOOKUP(A1, B1:E5, 2, FALSE) Horizontal lookup
INDEX(B:B, 5) Return 5th value in column B
MATCH(A1, B:B, 0) Find position of A1 in column B
INDEX(C:C, MATCH(A1, B:B, 0)) Flexible lookup alternative

SUMIF & COUNTIF

SUMIF(A:A, ">100", B:B) Sum B where A > 100
COUNTIF(A:A, ">100") Count cells > 100
AVERAGEIF(A:A, ">100", B:B) Average B where A > 100
SUMIFS(C:C, A:A, ">100", B:B, "<50") Multiple criteria sum
COUNTIFS(A:A, ">100", B:B, "<50") Multiple criteria count

Financial Functions

PMT(5%/12, 30*12, 200000) Loan payment calculation
FV(rate, nper, pmt, pv) Future value
PV(rate, nper, pmt, fv) Present value
NPV(rate, value1, value2, ...) Net present value
IRR(values, guess) Internal rate of return

Error Functions

IFERROR(formula, "Error") Handle any error
IFNA(VLOOKUP(...), "Not found") Handle #N/A errors
ISERROR(formula) Test if formula has error
Common Excel Errors:
• #DIV/0! - Division by zero
• #N/A - Value not available
• #NAME? - Unrecognized formula
• #REF! - Invalid cell reference
• #VALUE! - Wrong data type

Window Operations

Ctrl + Tab Switch between workbooks
Ctrl + F6 Switch workbook windows
F6 Move between panes
Ctrl + F9 Minimize workbook
Ctrl + F10 Maximize/restore window

Excel Tips

Productivity Tips:
• Use Ctrl+1 for quick formatting
• F4 to repeat last action
• Alt+= for quick sum
• Ctrl+Shift+L for instant filters
• F12 for quick Save As
Formula Best Practices:
• Use absolute references ($) when needed
• Name your ranges for clarity
• Use IFERROR to handle errors gracefully
• Document complex formulas with comments
• Test formulas with sample data first
