Matplotlib Visualization

Complete guide to plotting and data visualization with Matplotlib and Seaborn

Import & Setup

import matplotlib.pyplot as plt Import matplotlib
import numpy as np Import NumPy for data
import seaborn as sns Import Seaborn
%matplotlib inline Jupyter notebook display
plt.style.use('seaborn-v0_8') Set plot style

Basic Line Plots

plt.plot(x, y) Basic line plot
plt.plot(x, y, 'r--') Red dashed line
plt.plot(x, y, marker='o') Line with circle markers
plt.plot(x, y, linewidth=3) Set line width
plt.plot(x, y, color='#2E86AB') Custom color (hex)
plt.plot(x, y, alpha=0.7) Set transparency

Plot Labels & Titles

plt.xlabel('X axis') X-axis label
plt.ylabel('Y axis') Y-axis label
plt.title('Plot Title') Plot title
plt.legend() Show legend
plt.legend(['Line 1', 'Line 2']) Custom legend labels
plt.grid(True) Show grid
plt.grid(True, alpha=0.3) Transparent grid

Figure Management

plt.figure() Create new figure
plt.figure(figsize=(10, 6)) Set figure size
plt.show() Display plot
plt.savefig('plot.png') Save plot to file
plt.savefig('plot.png', dpi=300) Save with high DPI
plt.close() Close current figure
plt.tight_layout() Adjust spacing

Scatter Plots

plt.scatter(x, y) Basic scatter plot
plt.scatter(x, y, c=colors) Color-coded points
plt.scatter(x, y, s=sizes) Variable point sizes
plt.scatter(x, y, alpha=0.6) Transparent points
plt.colorbar() Add colorbar

Bar Charts

plt.bar(x, height) Vertical bar chart
plt.barh(y, width) Horizontal bar chart
plt.bar(x, height, color='skyblue') Colored bars
plt.bar(x, height, edgecolor='black') Bar edges
plt.bar(x, height, width=0.8) Set bar width

Histograms

plt.hist(data) Basic histogram
plt.hist(data, bins=30) Set number of bins
plt.hist(data, bins=np.arange(0, 100, 5)) Custom bin edges
plt.hist(data, alpha=0.7) Transparent histogram
plt.hist(data, density=True) Normalize to probability
plt.hist(data, cumulative=True) Cumulative histogram

Box Plots & Violin Plots

plt.boxplot(data) Basic box plot
plt.boxplot([data1, data2, data3]) Multiple box plots
plt.violinplot(data) Violin plot
plt.boxplot(data, labels=['Group 1', 'Group 2']) Labeled box plots

Subplots

fig, ax = plt.subplots() Create figure with axes
fig, axes = plt.subplots(2, 2) 2x2 subplot grid
fig, axes = plt.subplots(2, 2, figsize=(12, 10)) Subplots with size
ax.plot(x, y) Plot on specific axes
axes[0, 1].set_title('Top Right') Set subplot title
plt.subplot(2, 2, 1) Select subplot (alternative)

Axis Customization

plt.xlim(0, 10) Set x-axis limits
plt.ylim(-1, 1) Set y-axis limits
plt.xticks(range(0, 11, 2)) Set x-axis ticks
plt.xticks(rotation=45) Rotate x-axis labels
plt.xscale('log') Logarithmic x-axis
plt.yscale('log') Logarithmic y-axis

Annotations & Text

plt.text(x, y, 'text') Add text at coordinates
plt.annotate('label', xy=(x, y)) Annotate point
plt.annotate('label', xy=(x, y), xytext=(x2, y2)) Annotate with offset
Annotation with arrow:
plt.annotate('Maximum', xy=(x, y),
    xytext=(x+1, y+0.5),
    arrowprops=dict(arrowstyle='->'))

Seaborn Statistical Plots

sns.scatterplot(data=df, x='col1', y='col2') Scatter plot with data
sns.lineplot(data=df, x='x', y='y') Line plot with data
sns.barplot(data=df, x='cat', y='val') Bar plot with data
sns.boxplot(data=df, x='group', y='value') Box plot with groups
sns.violinplot(data=df, x='group', y='value') Violin plot

Seaborn Advanced

sns.heatmap(df.corr()) Correlation heatmap
sns.heatmap(data, annot=True) Heatmap with values
sns.pairplot(df) Pairwise relationships
sns.regplot(data=df, x='x', y='y') Regression plot
sns.distplot(data) Distribution plot
sns.jointplot(data=df, x='x', y='y') Joint distribution

Color Maps & Palettes

plt.cm.viridis Viridis colormap
plt.cm.plasma Plasma colormap
plt.cm.coolwarm Cool-warm colormap
sns.color_palette('husl', 8) Seaborn color palette
sns.set_palette('Set1') Set default palette

3D Plotting

from mpl_toolkits.mplot3d import Axes3D Import 3D toolkit
ax = fig.add_subplot(111, projection='3d') Create 3D axes
ax.plot3D(x, y, z) 3D line plot
ax.scatter3D(x, y, z) 3D scatter plot
ax.plot_surface(X, Y, Z) 3D surface plot

Styling & Themes

plt.style.use('default') Default style
plt.style.use('ggplot') ggplot style
plt.style.use('dark_background') Dark theme
sns.set_style('whitegrid') Seaborn white grid
sns.set_style('darkgrid') Seaborn dark grid
sns.set_context('paper') Set plot context

Interactive Features

plt.ion() Turn on interactive mode
plt.ioff() Turn off interactive mode
plt.draw() Redraw current figure
plt.pause(0.1) Pause for animation

Common Plot Types

Line plot with multiple series:
plt.plot(x, y1, label='Series 1')
plt.plot(x, y2, label='Series 2')
plt.legend()
plt.show()
Stacked bar chart:
plt.bar(x, y1, label='Group 1')
plt.bar(x, y2, bottom=y1, label='Group 2')
plt.legend()
Fill between areas:
plt.fill_between(x, y1, y2, alpha=0.3)
plt.plot(x, y1)
plt.plot(x, y2)

Visualization Tips

Best Practices:
• Use appropriate chart types for your data
• Keep plots simple and clear
• Choose colors carefully for accessibility
• Always label axes and add titles
• Use consistent styling across plots
Performance Tips:
• Use plt.ioff() for batch plotting
• Close figures with plt.close() to save memory
• Use rasterized=True for complex plots
• Consider data sampling for large datasets
• Use appropriate DPI for save quality
