Arduino IDE Programming

Arduino IDE shortcuts, programming syntax, board configurations, and microcontroller development commands

IDE Shortcuts

Ctrl+R Compile/Verify sketch
Ctrl+U Upload sketch to board
Ctrl+Shift+M Open Serial Monitor
Ctrl+Shift+L Open Serial Plotter
Ctrl+N New sketch
Ctrl+O Open sketch
Ctrl+S Save sketch
Ctrl+T Auto format code

Basic Structure

void setup() {} Initialization function, runs once
void loop() {} Main loop function, runs repeatedly
#include <library.h> Include external library
#define PIN_NAME 13 Define constant

Digital I/O

pinMode(pin, mode) Set pin mode (INPUT, OUTPUT, INPUT_PULLUP)
digitalWrite(pin, value) Write digital value (HIGH/LOW)
digitalRead(pin) Read digital value from pin
digitalToggle(pin) Toggle digital pin state

Analog I/O

analogRead(pin) Read analog value (0-1023)
analogWrite(pin, value) Write PWM value (0-255)
analogReference(type) Set analog reference voltage

Serial Communication

Serial.begin(baudrate) Initialize serial communication
Serial.print(data) Print data to serial
Serial.println(data) Print data with newline
Serial.available() Check if data available
Serial.read() Read serial data
Serial.readString() Read string from serial

Time Functions

delay(ms) Pause execution for milliseconds
delayMicroseconds(us) Pause execution for microseconds
millis() Get milliseconds since program started
micros() Get microseconds since program started

Math Functions

map(value, fromLow, fromHigh, toLow, toHigh) Map value from one range to another
constrain(x, a, b) Constrain value between a and b
abs(x) Absolute value
min(x, y) Minimum of two values
max(x, y) Maximum of two values
pow(base, exponent) Power function
sqrt(x) Square root

Data Types

boolean True or false value
byte 8-bit unsigned integer (0-255)
int 16-bit signed integer (-32,768 to 32,767)
long 32-bit signed integer
float 32-bit floating point number
double Same as float on Arduino
char Single character
String String object

Control Structures

if (condition) {} Conditional execution
if (condition) {} else {} If-else statement
for (int i=0; i<10; i++) {} For loop
while (condition) {} While loop
do {} while (condition) Do-while loop
switch (var) { case 1: break; } Switch statement

Interrupts

attachInterrupt(pin, function, mode) Attach interrupt to pin
detachInterrupt(pin) Remove interrupt from pin
interrupts() Enable interrupts
noInterrupts() Disable interrupts

Common Libraries

#include <Servo.h> Servo motor control library
#include <SoftwareSerial.h> Software serial communication
#include <Wire.h> I2C communication library
#include <SPI.h> SPI communication library
#include <EEPROM.h> EEPROM read/write library
#include <LiquidCrystal.h> LCD display library
