Common Workflows

Basic File Management

  1. Navigate: cd /path/to/directory
  2. List files: ls -la
  3. Create directory: mkdir new_folder
  4. Copy files: cp source destination
  5. Set permissions: chmod 755 file
  6. Check disk space: df -h

Log Analysis

  1. View log: tail -f /var/log/system.log
  2. Search patterns: grep "ERROR" logfile
  3. Count occurrences: grep -c "pattern" file
  4. Sort by frequency: sort | uniq -c | sort -nr
  5. Filter by date: awk '/Jan 15/ {print}' log

Remote File Transfer

  1. Generate SSH key: ssh-keygen -t rsa
  2. Copy public key: ssh-copy-id user@host
  3. Connect: ssh user@host
  4. Transfer file: scp file user@host:/path/
  5. Sync directories: rsync -av local/ remote:

Pro Tips

Tab Completion: Use Tab to auto-complete file names, commands, and paths
Command History: Ctrl+R for reverse search through command history
Wildcards: Use * for multiple characters, ? for single character
Man Pages: Use 'man command' to read manual pages for any command
Screen/Tmux: Use screen or tmux for persistent terminal sessions
Piping Power: Chain commands with | to create powerful one-liners
Sudo Usage: Use 'sudo !!' to run previous command as administrator
Hidden Files: Files starting with . are hidden; use ls -a to show them
macOS Spotlight: Use mdfind "search term" for Spotlight search from terminal
Quick Look: Use qlmanage -p file to preview files in terminal
