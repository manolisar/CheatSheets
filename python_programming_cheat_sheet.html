Python Programming Cheat Sheet

Essential Python syntax, built-in functions, data structures, and common patterns

Basic Syntax

Variables & Assignment

x = 5 Integer assignment
name = "John" String assignment
is_valid = True Boolean assignment
x, y = 1, 2 Multiple assignment
x += 1 Increment operator
x *= 2 Multiply and assign

Data Types

int() Integer conversion
float() Float conversion
str() String conversion
bool() Boolean conversion
type(x) Get variable type
isinstance(x, int) Check type

Strings

String Methods

s.upper() Convert to uppercase
s.lower() Convert to lowercase
s.strip() Remove whitespace
s.split(',') Split string by delimiter
','.join(list) Join list into string
s.replace('old', 'new') Replace substring
s.find('sub') Find substring index
len(s) String length

String Formatting

f"Hello {name}" f-string formatting
"Hello {}".format(name) .format() method
"Hello %s" % name % formatting
f"{x:.2f}" Format to 2 decimals

Lists

List Operations

list = [1, 2, 3] Create list
list.append(4) Add to end
list.insert(0, 0) Insert at index
list.remove(2) Remove first occurrence
list.pop() Remove last element
list.pop(0) Remove element at index
len(list) List length
list.sort() Sort in place
sorted(list) Return sorted copy

List Slicing

list[0] First element
list[-1] Last element
list[1:3] Slice from index 1 to 2
list[:3] First 3 elements
list[2:] From index 2 to end
list[::-1] Reverse list

Dictionaries

Dictionary Operations

d = {'a': 1, 'b': 2} Create dictionary
d['c'] = 3 Add/update key
d.get('a', 0) Get with default
del d['a'] Delete key
d.keys() Get all keys
d.values() Get all values
d.items() Get key-value pairs
'a' in d Check if key exists

x if condition else y Ternary operator

Loops

for i in range(5): For loop with range
for item in list: Iterate over list

while condition: While loop
break Exit loop
continue Skip iteration

Common Workflows

Read CSV File

  1. import csv
  2. with open('file.csv') as f:
  3.   reader = csv.reader(f)
  4.   for row in reader:
  5.     print(row)

API Request

  1. import requests
  2. response = requests.get(url)
  3. if response.status_code == 200:
  4.   data = response.json()
  5.   process(data)

Data Processing

  1. data = [1, 2, 3, 4, 5]
  2. filtered = [x for x in data if x > 2]
  3. squared = [x**2 for x in filtered]
  4. result = sum(squared)
