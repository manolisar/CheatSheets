## Troubleshooting and Performance

### Common Issues
```bash
Memory Management:
- Purge memory regularly (Edit > Purge)
- Enable disk cache (Preferences > Media & Disk Cache)
- Adjust RAM allocation (Edit > Preferences > Memory)

Performance Tips:
- Use proxies for high-res footage
- Pre-compose complex animations
- Cache work area when possible
- Disable unused effects
- Use adjustment layers efficiently

GPU Acceleration:
- Enable Mercury GPU acceleration
- Update graphics drivers
- Use CUDA/OpenCL compatible cards
- Monitor GPU usage in Info panel
```
